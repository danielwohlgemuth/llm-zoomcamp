# LLM Zoomcamp

[LLM Zoomcamp](https://github.com/DataTalksClub/llm-zoomcamp) is free online course by [DataTalks.Club](https://datatalks.club/) to learn
- agentig retrieval-augmented generation
- function calling
- vector search
- embeddings
- orchestration
- evaluation
- monitoring

## Project: AWS Introduction Search

My project idea for this course is a search service that uses AWS introduction information as the knowledge base.
Since LLMs will have been trained extensively on AWS material, they are likely to respond directly instead of using the knowledge base.
To overcome that, the AWS names are replaced with a space-themed alternative.

In order to find and collect the introductions of AWS, the [AWS Pricing Calculator](https://calculator.aws/#/addService) list was used.
